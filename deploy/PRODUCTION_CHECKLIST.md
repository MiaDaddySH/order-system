# Production Go-Live Checklist

## 1. 配置与密钥

- [ ] `SPRING_PROFILES_ACTIVE=prod`
- [ ] `DB_URL`、`DB_USERNAME`、`DB_PASSWORD` 已正确注入
- [ ] `JWT_SECRET` 为高强度随机密钥（至少 32 字符）
- [ ] `CORS_ALLOWED_ORIGINS` 仅包含真实前端域名

## 2. 数据库与迁移

- [ ] PostgreSQL 可用且账号权限最小化
- [ ] Flyway 迁移执行成功
- [ ] 数据库备份策略已配置（全量 + 增量）
- [ ] 恢复演练至少完成一次

## 3. 网络与证书

- [ ] 80 端口跳转到 443
- [ ] TLS 证书有效且自动续期可用
- [ ] 反向代理只暴露必要路径和端口
- [ ] 防火墙只开放必要端口

## 4. 应用验证

- [ ] `/actuator/health` 正常
- [ ] `/v3/api-docs` 正常
- [ ] 注册 -> 登录 -> `/users/me` 链路通过
- [ ] 限流与鉴权行为符合预期

## 5. 可观测性与运维

- [ ] 错误日志可检索，敏感信息不落日志
- [ ] 关键指标可观测（请求量、错误率、延迟）
- [ ] 告警规则已配置并触发测试通过
- [ ] 回滚方案已准备（镜像版本/数据库回滚策略）
